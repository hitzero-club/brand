# HitZero Brand Guide

Guía de marca interactiva de HitZero. Contiene paleta de colores, tipografía, uso de logo, tono de voz y análisis de pricing.

## Stack

- HTML + CSS + vanilla JS
- Zero build step
- Deploy: GitHub Pages

## Estructura

```
├── index.html          # Brand guide (single page)
├── assets/             # Logo SVG, PNG y recursos de marca
│   ├── logo.svg
│   ├── logo.png
│   ├── logo-white.svg
│   └── ...
└── .github/workflows/  # Deploy automático a GitHub Pages
```

## Desarrollo local

Abrí `index.html` en el navegador. No necesita servidor ni build.

## Assets disponibles

Los assets de marca (logos, iconos) se van agregando en `/assets/` para fácil acceso y descarga.
